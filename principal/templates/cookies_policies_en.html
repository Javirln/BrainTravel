What is a Cookie?

A 'cookie' is a small text file that a website stores on browser user. Cookies make using and navigating a website and are essential for Internet operation, providing many advantages in service delivery interactive.

Cookies are used, for example, to manage the user's session (reducing the number of times that you must insert your user data), improving the services offered, or to adapt the contents of a web page to your preferences.

Cookies also can be 'proper', managed by the domain to which the user accesses and requesting a service (in this case the home owned by BrainTravel Co., Inc.) or 'third party cookies', sent to the user equipment from a domain accessed differently.

Common cookies in the webpages of braintravel.com

  • Session Cookie ('sessionid'): Used to uniquely identify each user accessing the application. This cookie is necessary to use these websites, so if the user decides to delete this cookie from your device, the web site may not function correctly.
  • CSRF token cookie ('csrftoken'): It is used as a security measure to prevent various attacks.
  • Aceptance cookie ('cp-enabled'): Used to remember if the user has already accepted the cookie policy of the site.
